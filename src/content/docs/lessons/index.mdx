These are One Degree North's training modules, organized by subteam. New
members: start with your subteam's intro lesson in the sidebar. Use the
search bar (top of the page) to find anything.

## For lesson authors

Anyone on the team can add lessons — no coding required:

1. Open the [content editor](https://frc-website-zeta.vercel.app/keystatic)
   and sign in with GitHub. (You need to be in the team's **FRC Robotics**
   GitHub group — ask a team lead to add you.)
2. Pick your subteam, click **Add**, and write your lesson. Headings, links,
   and code blocks are in the toolbar; images can be dragged straight into
   the editor.
3. Make the **slug** start with a number to control its order in the
   sidebar, like `03-sensors` (the editor shows a hint for this).
4. Save. The site updates automatically within a couple of minutes.

Each subteam's intro lesson doubles as a formatting example — copy what it
does.
