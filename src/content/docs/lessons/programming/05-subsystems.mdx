**Agenda:**

* What is a subsystem? Use a company analogy — the 'intake department' owns its conveyor belt and doesn't let anyone else touch it directly.
* Create an Intake subsystem extending SubsystemBase. Add a TalonFX field. Show how the subsystem exposes methods like runIntake() and stopIntake().
* Intro to CTRE TalonFX (conceptual only): what is a motor controller? Device ID, how it connects over CAN.
* Activity: write an Elevator subsystem stub — just the class skeleton with a few methods.
